# 🔍 Smart Lost & Found (SmartL&F)

**SmartL&F** is an intelligent, AI-powered campus lost and found system designed specifically for the students and faculty of NIT Silchar. It streamlines the process of reporting, finding, and claiming lost items using smart matching, real-time dashboards, and secure verification.

*Originally built for **CSS Hacks**.*

---

## ✨ Key Features

* 🌓 **Light & Dark Mode:** Seamless theme switching with persistent local storage preferences.
* 🤖 **AI-Powered Matching (Simulated):** Intelligently scores reported items against found items based on description, location, and category.
* 📝 **Comprehensive Reporting:** Dedicated forms for reporting lost or found items, complete with image uploads, location tracking, institute email, and mobile number integration.
* 🔎 **Live Search & Filters:** Browse items with real-time text search and filter by status (Lost/Found), category, and campus location.
* 📊 **Interactive Campus Dashboard:** * Animated live statistics counters.
    * Recent activity feed tracking reports and matches.
    * Weekly overview bar charts.
    * Loss location heatmaps/progress bars.
* 🔐 **Secure Claiming:** Modal interface for verifying identity via Scholar ID and OTP before an item can be claimed.
* 📱 **Fully Responsive:** Mobile-first design with a custom hamburger menu and optimized layouts for all screen sizes.

---

## 🛠️ Tech Stack

This project is built as a lightweight, lightning-fast Single Page Application (SPA) using purely native web technologies—no heavy frameworks required.

* **Structure:** HTML5
* **Styling:** CSS3 (Custom Properties/Variables, Flexbox, CSS Grid, Keyframe Animations, Glassmorphism effects)
* **Logic:** Vanilla JavaScript (ES6+)
* **Typography:** [Google Fonts](https://fonts.google.com/) (Outfit & DM Sans)
* **Icons:** [FontAwesome 6.5.0](https://fontawesome.com/)
